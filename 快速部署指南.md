# 🚀 快速部署指南 - 3 分钟上线

## ⚡ 推荐方案：Vercel 一键部署（最简单）

### 步骤 1：访问 Vercel

打开 https://vercel.com/new

### 步骤 2：导入 GitHub 仓库

1. 点击 **"Import Git Repository"**
2. 选择你的仓库：`Sales-Champion-Cloud-Group`
3. 点击 **"Import"**

### 步骤 3：配置项目

**Framework Preset**: Next.js（自动检测）

**Root Directory**: 点击 **"Edit"**，输入 `frontend`

**Build and Output Settings**:
- Build Command: `npm run build`
- Output Directory: `.next`

### 步骤 4：添加环境变量

点击 **"Environment Variables"**，添加：

| Name | Value |
|------|-------|
| `DATABASE_URL` | `sqlite:///./sales_champion.db` |
| `JWT_SECRET` | `your-secret-key-change-in-production` |
| `NEXT_PUBLIC_API_BASE_URL` | `/api/v1` |

### 步骤 5：部署

点击 **"Deploy"**，等待 2-3 分钟完成。

### ✅ 完成！

Vercel 会给你一个 URL：`https://your-app.vercel.app`

---

## 🌐 备选方案：Netlify 部署

### 步骤 1：注册 Netlify

访问 https://netlify.com

### 步骤 2：连接 GitHub

1. 点击 **"Add new site"** → **"Import an existing project"**
2. 选择 **GitHub**
3. 授权并选择你的仓库

### 步骤 3：配置构建

- **Base directory**: `frontend`
- **Build command**: `npm run build`
- **Publish directory**: `.next`

### 步骤 4：设置环境变量

在 **Site settings** → **Environment variables** 添加：

- `NEXT_PUBLIC_API_BASE_URL`: 你的后端地址

### 步骤 5：部署

点击 **"Deploy site"**

## 🔧 高级方案：Docker 部署（完整控制）

### 本地测试

```bash
# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 访问 http://localhost:3000
```

### 部署到云服务器

1. 购买云服务器（阿里云/腾讯云/AWS）
2. 安装 Docker
3. Clone 仓库
4. 运行 `docker-compose up -d`
